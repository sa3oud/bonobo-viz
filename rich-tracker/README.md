# Rich Crypto Tracker

A professional Go full-stack application featuring a real-time price stream and secure API access.

## Features
- **Real-time Streaming:** Uses Server-Sent Events (SSE).
- **Professional Layout:** Follows the Go Standard Project Layout.
- **Security:** Custom Auth Middleware via API Key.
- **Frontend:** Responsive Tailwind CSS and Chart.js dashboard.

## How to Run
1. Set your API Key: `export API_KEY=your_secret`
2. Run the server: `go run cmd/server/main.go`
3. Visit `http://localhost:8080`
